# One-Page Project Plan

## Project

UserSteer Recommender Bench: Information retrieval for Giles-Aligned Research

## Professor Fit

This project is aligned with C. Lee Giles's listed research area: **Information retrieval, search engines, digital libraries, deep learning**.

## Problem

Current AI systems in this area often look strong on aggregate benchmarks while hiding failures in grounding, distribution shift, calibration, data leakage, recovery, user control, or scientific validity. This project turns that broad weakness into a concrete reproducible artifact.

## Proposed Artifact

A transparent ranking or recommendation system with user-controllable evidence.

## First Experiment

Build a BEIR/MovieLens-style baseline and add counterfactual explanations or preference controls with a small evaluation protocol.

## Evaluation

Use metrics and analysis targeted to the artifact: reproducible baselines, ablations, failure slices, cost/runtime, and clear limitations.

## Why This Helps A Lab

- It creates reusable infrastructure, not just a one-off result.
- It can expose failure cases that become publishable research questions.
- It gives the professor a low-risk way to evaluate your research taste, execution, and honesty.
